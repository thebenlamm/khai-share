<objective>
Fix the export name mismatch that silently disables regression detection, and correct the MCP docstring field names.

Purpose: src/routes/baselines.js exports `module.exports.manager` but src/routes/api.js destructures `{ baselineManager }`, which resolves to `undefined`.
