<!-- REST API endpoints the MCP tools will call (from src/routes/baselines.js): -->

POST /api/baselines
  Body: { testId: string, thresholds?: { responseTime?: number, pageLoadTime?: number } }
  Returns 201: { success: true, data: { id, site, account, sourceTestId, createdAt, updatedAt, thresholds, snapshot } }
  Returns 409 if baseline already exists for site+account

GET /api/baselines?site=optional
  Returns 200: { success: true, data: { baselines: [...] } }
  List items include metadata only (snapshot.pages excluded, pageCount included)

GET /api/baselines/:id
  Returns 200: { success: true, data: { id, site, account, ..., snapshot: { pages: [...] } } }
  Returns 404 if not found

PUT /api/baselines/:id
  Body: { testId: string }
  Returns 200: { success: true, data: { ...updated baseline } }
  Returns 404 if baseline not found

DELETE /api/baselines/:id
  Returns 200: { success: true, data: { deleted: true } }
  Returns 404 if not found

<!-- MCP client module (from khai_mcp/client.py): -->
client.get(path) -> dict
client.post(path, data) -> dict
client.put(path, data) -> dict
client.delete(path) -> dict

   - Params: site (str | None = None)

3. **khai_baseline_get** (readOnlyHint=True):
   - Params: baseline_id (str)
   - Calls: client.get(f"/api/baselines/{baseline_id}")

4. **khai_baseline_update** (destructiveHint=True):
   - Params: baseline_id (str), test_id (str)
   - Calls: client.put(f"/api/baselines/{baseline_id}", {"testId": test_id})
   - Unwrap and return updated baseline

5. **khai_baseline_delete** (destructiveHint=True):
   - Params: baseline_id (str)
   - Calls: client.delete(f"/api/baselines/{baseline_id}")
