ENDPOINTS TO IMPLEMENT:

POST /api/watches — Create a watch
- Body: { site, account, url, schedule, selector?, webhookUrl?, enabled? }
- Validation: site, account, url, schedule are required — return 400 if missing
- Call manager.addWatch(body) — it throws on invalid cron expression
- Wrap in try/catch: cron validation errors return 400; other errors use errorHandler
- Return 201 with ok(watch)

GET /api/watches — List all watches
- Return ok({ watches: manager.listWatches() })

GET /api/watches/:id — Get a single watch
- safeId(req.params.id)
- const watch = manager.getWatch(req.params.id)
- If null: return 404 with fail('Watch not found')
- Return ok(watch)

PUT /api/watches/:id — Update a watch
- safeId(req.params.id)
- Only pass allowed fields: { url, selector, schedule, webhookUrl, enabled }
- Build updates object from req.body (only include keys that are present — don't overwrite with undefined)
- Call manager.updateWatch(id, updates) — it throws on invalid cron
- If null returned: 404
- Wrap in try/catch: cron validation errors return 400; other errors use errorHandler
- Return ok(updated)

DELETE /api/watches/:id — Delete a watch
- safeId(req.params.id)
- const removed = manager.removeWatch(req.params.id)
- If false: 404
- Return ok({ deleted: true })

GET /api/watches/:id/history — Get run history
- safeId(req.params.id)
- const watch = manager.getWatch(req.params.id)
- If null: 404
- const limit = parseInt(req.query.limit) || 20
- const history = manager.getHistory(req.params.id, limit)
- Return ok({ watchId: req.params.id, history })

POST /api/watches/:id/run — Trigger immediate manual run
- safeId(req.params.id)
- const watch = manager.getWatch(req.params.id)
- If null: 404
- Fire manager._runWatch(watch.id) WITHOUT awaiting (fire-and-forget, same as scheduled runs)
- Return 202 with ok({ message: 'Run triggered', watchId: watch.id })
